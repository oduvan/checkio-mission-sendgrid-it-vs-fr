How to start here?
You should use /geo/stats in order to get information about clicks and opens grouped by country. The API contains two-digit country count; this is what we are looking for.
Can you show me how can I do it in Python?

Sure. This example shows statistics for the first 7 days of the year.

response = sg.client.geo.stats.get(query_params={
    'start_date':'2016-01-01',
    'end_date': '2016-01-07'
})
import json

print(json.loads(response.body))
What should I do with dates?
If the start date and the end date are the same you will receive information for one day only. And this is what what we are looking for. 
response = sg.client.geo.stats.get(query_params={
    'start_date':str_date,
    'end_date': str_date
})
What can I do with that data after?

        {
         'type': 'country',
         'name': 'AU',
         'metrics': {'clicks': 0, 'opens': 31, 'unique_clicks': 0, 'unique_opens': 22}

You should go through the list 'stats' and find a value of 'name' with maximum 'unique_clicks'.
